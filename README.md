This is a Habit Tracker  Project which is built using Nodejs, Ejs , CSS, Javascript


## Features
- Add Habit <br/>
- Each Habit Track Weekly<br/>
  - Mark Habit as Done
  - Mark as Undone
  - Toggle to Default - No action setting
  - Delete a habit

## Packages
- body-parser <br/>
- dotenv<br/>
- ejs<br/>
- express<br/>
- mongoose<br/>
- nodemon<br/>
